How can I create an interactive data visualization that updates as users scroll through?

By Vince Egalla

(Note: If the visualisation doesn't fit your screen, try zooming out with ctrl + - / Cmd + -.)

This work is inspired by and draws from the work of Cuthbert Chow, Jim Vallandingham, and Michael Bostock.

I am inspired to provide a tutorial on this technique to assist peers in my Master's of Data Science for Public Policy at Georgetown University, who may be interested in creating data story websites.

A basic understanding of HTML, CSS, and JavaScript may be necessary to implement this technique. However, even without knowledge with these languages, I hope to provide a template for peers to learn from and utilize.

Introduction

The four important components of this template are:

  • index.html
  • scroller.js
  • section.js
  • style.css

index.html contains the website metadata and text content

scroller.js adds the scrolling-updating component to the website

section.js handles data input and draws the visualization using D3.js

style.css provides additional styling details

Over the course of this tutorial, I'll be progressively building a network visualization.
